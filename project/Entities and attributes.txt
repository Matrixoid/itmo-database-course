Сущности и их атрибуты:
Characters - персонажи доступные в игре
	1. Id - id персонажа
	2. Name - имя персонажа
	3. Level - текущий уровень персонажа
	4. Rank - текущий ранг персонажа
Ranks - возможные ранги персонажей и оружия
	1. Rank - ранг
Resourses - виды ресурсов, существующих в игре и необходимых для прокачки персонажей или оружия
	1. Id - id ресурса
	2. Name - название ресурса
	3. Rarity - редкость ресурса(3, 4 или 5 звёзд)
	4. ResourseClassId - класс ресурса
Inventory - инвентарь пользователя
	1. ResourseId - id ресурса
	2. ResourseCount - количество ресурса
ResourseClasses - ресурсы будем делить на 5 классов по методу их добывания и те, которые мы к первым четырём классам отнести не сможем(книги опыта или камни возвышения)
	1. Id - id класса
	2. Name - название класса
ResourseCountForCharacterElevations
	1. ElevationNumber - номер возвышения(всего их 6)
	2. ResourseClassId - id класса ресурса
	3. ResourseCount - количество ресурса
CharacterLevels - таблица необходимого опыта для прокачки каждого уровня персонажа(их 90)
	1. Level - уровень
	2. Experience - необходимый опыт
Weapons - оружие доступное в игре
	1. Id - id оружия
	2. Name - название оружия
	3. Level - текущий уровень оружия
	4. Rank - текущий ранг оружия
ResourseCountForWeaponElevations
	1. ElevationNumber - номер возвышения(всего их 6)
	2. ResourseClassId - id класса ресурса
	3. ResourseType - для возвышения оружия требуется 3 вида ресурсов, два из которых одного класса. Чтобы их различать введём тип
	4. ResourseCount - количество ресурса
WeaponLevels - таблица необходиомго опыта для прокачки каждого уровня оружия(их 90)
	1. Level - уровень
	2. Experience - необходимый опыт
